# Hiragana Trivia

Best for learning hiragana.

## Run

```
python nihongo.py
```

First, you need to choose question from あ(a) to ん(n)

Now you can't start to answer the question

```
What is this :く
ku
Correct!!!
What is this :そ
se
Wrong, please try again
```
